Audio Panning Demo (Tested in Chrome Only)

Use the w, a, s, d keys or arrow keys to move the background sound around in 3D. The q key toggles the background music on and off. The r and l keys growl right or left respectively.

This is a demo of the audio API stereo capabilities, so you will need to have the sound on and headphones or stereo speakers for it to have any effect. There is no visible demo (x,y,z positions are reported in the console, though).
